Winger: A Window Manager

Quick Guide

Last updated v1.4.2

The Winger Panel

Click on the Winger button on your browser toolbar, or press Alt+W, to open the pop-up panel.

Tip: You can move browser toolbar buttons onto the tab strip – a reasonable spot for tab/window-related actions! To move buttons around, right-click the toolbar and click Customize to enter Customize Firefox mode.

Every window is represented by a row, showing its name and tab count.

The topmost row represents the currently focused window (current window).

Below it is the panel omnibox.

If there are other windows, they are listed below the omnibox; the first row representing the most recently focused (last seen) window, followed by the next most recently focused window, and so on.

Below the list is the status bar showing the total number of tabs and windows open.

Lastly at the bottom is the panel toolbar.

Common Actions

• Switch Window

Click on a row to switch (change focus) to the corresponding window.

• Send Tab

The send button "sends" the selected tab(s) to the corresponding window without leaving the current window.

Ctrl+switch becomes a send. Thus, you can also ctrl+click a row to send tabs to that window.

• Bring Tab

The bring button switches to the corresponding window while "bringing" the selected tab(s) along. Essentially switch and send combined.

Shift+switch becomes a bring. Thus, you can also shift+click a row to bring tabs to that window.

Private windows

Winger can list and interact with private windows if it is allowed private window access. Check .

Private windows are marked with purple names.

Tabs technically cannot move between normal and private windows, but Winger can perform a close equivalent: send and bring will close tabs in the current window and reopen them in the destination window. Buttons that will perform this are shown purple.

  • Pinned, unloaded (a.k.a. suspended, discarded) and reader-view tab states will be preserved.

  • Other tab and page states – including tab history, scroll position and unsubmitted form data – will be lost in the process.

Pinned tabs

Send and bring can work on pinned tabs if enabled in .

Name Windows

The edit button lets you name/rename a window.

You can also edit the current window's name by double-clicking it.

When the panel is in Edit Mode, you can type in a new name and then use the following keys:

  • Enter to save the currently edited name and exit Edit Mode.

  • Up or Down to save the currently edited name and edit another window name.

  • Esc to close the panel, exiting Edit Mode without saving any changes to the currently edited name.

Names must be unique and may not begin with a forward slash ("/").

Winger assigns every window a default name (Window 1, Window 2 and so on), visible when a window has no user-given name.
Unlike user-given names, default names are temporary and not guaranteed to persist across browser sessions.

Naming tips

  • In Edit Mode, the status bar will show the title of the targeted window's active tab, to help you identify any windows that are not visible.

  • You can include emojis in names. Windows and Mac have built-in emoji pickers (Win+. and Cmd+Ctrl+Space respectively). There are also emoji extensions.

Where a window name appears

  • Title bar. Contents of the title bar also show up in your operating system's task manager or switcher (e.g. while Alt-Tabbing), even if you've set Firefox's title bars to be hidden.

  • Tooltip that appears when you hover your pointer over the Winger button .

  • Button badge on the Winger button (showing the first few characters of a name), if enabled in .

  • And of course, open the Winger panel any time to see the current window's name in bold.

Stash Windows NEW

If you want to close a window, but also be able to restore it at any time in the future, you can stash it. Stashed windows are stored as bookmark folders, which you can then unstash.

  • To use this feature, enable it in .

  • To stash the current window, open the panel and enter /stash in the omnibox.

    The window will close and an identically-named bookmark folder – containing bookmarks of tabs – will appear in the stash home.

  • To unstash a bookmark folder, right-click on it and click the Unstash context menu command.

    The folder will disappear and an identically-named window will open, containing the formerly-bookmarked tabs, in an unloaded state whenever possible.

    • Unstash also works on individual bookmarks. The command will remove the bookmark and open it as a new tab in the current window.

    • Unstash is available to all bookmarks and folders, not just within the stash home.

The stash home and stash folders

The stash home is where stash folders ("stashed windows") are stored. It can be changed in to be one of the three root folders – Bookmarks Toolbar, Bookmarks Menu or Other Bookmarks – or a "Stashed Windows" folder inside a root folder.

If the stash home folder contains at least one separator, Winger treats only folders after the last separator as stash folders, ignoring everything before it. It effectively allows for a stash home 'zone' set apart from other contents of the folder.

This matters for root folders, which tend to already contain bookmarks and folders for other purposes. Therefore if you designate a root folder without separators as your stash home, Winger will automatically add a separator.

You can manually move bookmark folders in and out of the stash home to include or exclude them as stash folders.

(Q: Why does it matter what is or isn't a stash folder? A: In a future update, the panel may also list "stashed windows".)

The stash feature is currently still being developed, building up from basic functionality first. Significant improvements to come!

Keyboard Navigation
& The Omnibox

Press Alt+W to open the panel. You can always check what the keyboard shortcut is by hovering your pointer over the Winger button .

(While Firefox's Find panel is open, Winger's default shortcut Alt+W conflicts with the Whole Words button shortcut, which takes precedence. Simply close the Find panel with Esc to use the shortcut as you intended.)

You can use the arrow keys to navigate the panel: Up and Down to select rows, Left and Right to select buttons. Press Enter or Space to activate a row or button.

You should also use Winger's omnibox – a multi-purpose, multi-talented input field – to vastly accelerate operating Winger by keyboard.

Primary omnibox powers

  • Start typing in a window name to filter the list of other windows.

  • Press Enter to switch to the first window listed below the omnibox.

    Recall that Ctrl+switch becomes a send and Shift+switch becomes a bring.

    Tip: Pressing either of these modifier keys while the omnibox is empty will hint what it does.

In summary, to swiftly switch or send tabs or bring tabs to a window via keyboard:

  1. Alt+W to open the panel.

  2. If the desired window row is not already immediately below the omnibox, start typing its name until it is.

  3. Enter OR Ctrl+Enter OR Shift+Enter.

Alternatively, use arrow keys to select the desired row or button in the panel.

Slash commands

Certain actions can be performed by typing one of these slash commands in the omnibox, followed by Enter. Slash commands auto-complete as you type.

/help or /h or just /
Open this help page
/settings or /s
Open Winger settings
/edit or /e
Enter Edit Mode to rename windows
/stash or /st
Stashes the current window

Tip: Customise the panel

Once you get comfortable enough with the omnibox, you have the option of removing any buttons you don't use.

Remove them all if you like a minimalist panel!

Other Windows Submenu

A submenu of other windows – a list of windows sorted by recently focused – becomes available in the following context menus when there is more than one window open.

Tab context menu

Right-click on a tab to get the Send Tab to Window menu command, which opens the submenu.

Click a window name to send the tab to the chosen window. (And as you might expect, Shift+click to bring the tab instead.)

The command is available to both selected and unselected tabs.

If you right-click on one among multiple selected tabs, the command will apply to all of them.

Link context menu

Right-click on a link in a page to get the Open Link in Window menu command, which opens the submenu.

Click a submenu item to open the link in the chosen window. (Shift+click will also switch to the window.)

Tips for Managing Large Tab/Window Counts

  • Use a tab discarding (a.k.a. unloading/suspending) extension to minimise memory usage on less-visited tabs and windows.

  • Divide and conquer with "window mitosis": split up a highly-populated window into several windows so you can focus on one set of tabs at a time. Hopefully this helps you work through them easier, and close them sooner.

  • If you like to proactively keep the number of windows under a certain number, you can always check the window count on the panel status bar. Also, the list of other windows starts to overflow and scroll once there are about 12 of them; this can be a cue you can use to start closing some windows.

  • If you are not ready to permanently close windows you hardly use, consider stashing them.

  • The "List all tabs" button, which appears whenever the tab strip overflows, opens a menu of every tab in the current window. Its shortcut is Ctrl+Shift+Tab.

    • Each menu item can be right-clicked, just like a tab, to open its own context menu. This is as close as Firefox currently gets to native vertical tabs!
