Winger: A Window Manager

A Complete Guide

Last updated v2.2.0

View latest release notes on the Winger addon page

Winger helps you manage multiple browser windows, no matter how many. With Winger you can name them, find/filter them, switch between them, move tabs between them, stash them away, and more.

First things first...

Your browser settings

In order for your windows and their names to persist across browser restarts: in Firefox Preferences, enable Open previous windows and tabs.

Some basics of working with tabs and windows

• Select multiple tabs

Ctrl+click on each tab you want to select, one by one.

Shift+click on a tab to select all the tabs between it and the currently focused tab, inclusive.

• Create a window

Ctrl+N to open a new window.

Ctrl+Shift+P to open a new private window.

Tear off tabs: With your pointer, drag-and-drop a tab off the tabs bar to create a new window containing the selected tabs.

Shift+click a link to open the link in a new window.

(Ctrl+click or middle-click opens it in a new tab.)

• Close a window

(Not the same as quitting the browser, which closes all windows.)

Click the × button at the top of a window to close it.

Ctrl+Shift+W to close the current browser window.

(Ctrl+W closes the selected tabs.)

Common shortcuts to close any window:

  • Alt+F4
  • Alt+Space C

The Winger panel

Click on the Winger button on your browser toolbar, or press F1, to open Winger's pop-up panel. (You can always check what the keyboard shortcut is by hovering your pointer over the Winger button.)

  • Tip: You can move browser toolbar buttons onto the tabs bar – a reasonable spot for tab/window-related buttons! To be able to move buttons around, right-click the toolbar and click Customize Toolbar.

Every window is represented by a row, showing its name and tab count.

Above the panel omnibox is the row representing the currently focused window (current window).

If there are other windows, they are listed below the omnibox; the first row representing the most recently focused (last seen) window, followed by the next most recently focused window, and so on.

At the bottom is the status bar showing the total number of tabs and windows open, and the help button.

Name windows / Edit mode

To enter edit mode, click on the current window's name, or press Up Enter from the omnibox.
(Alternatively, type /edit Enter in the omnibox to toggle edit mode on or off.)

In edit mode, all names become targetable and editable, and common actions are disabled.

To exit edit mode without closing the panel, press Enter while in a name field.

Names should be unique and may not start with a forward slash (/).

A valid window name is saved when you de-target it (by clicking or navigating away from it), or when you press Enter in a name field. Invalid names, such as duplicates, are reverted upon any save attempts.

To cancel and not save any changes to the currently targeted name, close the panel by clicking outside it or by pressing Esc. This also exits edit mode.

Common actions

Switch to window

Click on a row, or press Enter on a targeted row, to switch to that window.

Send tab

The send button "sends" the selected tabs to that window without leaving the current window.

Ctrl+switch becomes a send. Thus you can also Ctrl+click or Ctrl+Enter on a row to send tabs to that window.

Bring tab

The bring button switches to that window while "bringing" the selected tabs along. Essentially switch and send combined.

Shift+switch becomes a bring. Thus you can also Shift+click or Shift+Enter on a row to bring tabs to that window.

Private windows

Winger can list and interact with private windows if it is allowed private window access. Check your .

Tabs technically cannot move between normal and private windows, but Winger can perform a close equivalent: send/bring closes the selected tabs in the current window and reopens them in the destination window.

Send/bring buttons that will reopen tabs appear purple when targeted or hovered over.

Reopened tabs...

  • ...preserve any pinned and reader-view tab states, but not most other tab/page states such as tab history, scroll position and unsubmitted form data.

  • ...start out in an unloaded state (a.k.a. suspended, discarded) if not focused.

    • Reopening multiple tabs in an unloaded state ensures that such actions remain non-intensive and instantaneous, no matter how many tabs. Otherwise, loading large numbers of tabs all at once may briefly strain your system, potentially slowing your browser to a crawl.

  • ...with privileged URLs that Winger cannot directly open (e.g. local files, most about: pages) become temporary placeholder tabs. Placeholder tabs will inform you of a manual step required to properly restore them.

The omnibox

The omnibox is a multi-purpose, multi-talented field that vastly accelerates operating Winger by keyboard. Here's what you can do with it:

Perform actions on the topmost other window

Press Enter, Ctrl+Enter or Shift+Enter in the omnibox to switch, send or bring to the first window listed below it.

This means upon opening the panel, you can immediately perform any of the common actions on the last seen window via keyboard.

Find/filter windows

Looking for a window? Type its name.

Typing in something filters the list of other windows. This way you can change which window appears below the omnibox, pinpointing your destination. Alternatively, the now shortened window list can also be traversed with arrow keys more quickly.

Filtering works in edit mode too, for hunting down a window you want to rename.

Combine these two abilities to quickly, say, switch to another window by keyboard like so:

  1. F1 to open the panel.
  2. If the desired window is not immediately below the omnibox, start typing its name until it is.
  3. Enter.

Send-to-Window menu command

Tab context menu

Right-click on a tab to find Send to Window in the context menu, which shows a submenu listing all other windows. Then click on a window name to send the tab there. (Shift+click to bring instead.) This works whether or not the tab was selected.

If you had right-clicked on one of multiple selected tabs, the action will apply to all of them, as you would expect.

Link context menu

Right-click on a link in a page to find the same Send to Window menu command. Then click on a window name to open the link there. (Shift+click will also switch to the window.)

Stash windows

If you want to close a window, but also be able to restore it at any time, you can stash it. Stashed windows are stored as bookmark folders, which you can then unstash.

Stashing streamlines the multistep process of saving a window's tabs as bookmarks, into a newly-created folder you'd be manually naming, and then closing the window. Unstashing reverses the process just as smoothly.

  • To use this feature, enable it in .
  • To stash the current window, open the panel and type /stash Enter in the omnibox.
    The window will close and an identically-named bookmark folder – containing bookmarks of tabs – will appear in the stash home.

  • To unstash a bookmark folder, right-click on it and click the Unstash menu command.
    The folder will disappear and an identically-named window will open, containing the formerly-bookmarked tabs.

    • Unstash also works on individual bookmarks. The command will delete the bookmark and open it as a new tab in the current window.

    • Unstashed tabs are similar to reopened tabs: unloaded if possible, as placeholder tabs if necessary. This makes Unstash a superior bookmark folder command compared to the browser's native Open All Bookmarks.

  • "Non-destructive" stashing/unstashing is possible with the Shift key:

    • /stash Shift+Enter to keep the window from being closed.
    • Shift+click the Unstash menu command to keep the folder/bookmark from being deleted.

The stash home and stash folders

The stash home is where stash folders ("stashed windows") are saved. It can be changed in to be one of the three root folders – Bookmarks Toolbar, Bookmarks Menu or Other Bookmarks – or a "Stashed Windows" folder inside a root folder.

Handy tip

With Firefox's bookmark syncing, you can stash a window and then unstash it on another computer.

Tips for managing lots of tabs and windows

  • Divide and conquer with "window fission": split up a highly-populated window into several windows so you can focus on one set of tabs at a time. Hopefully this helps you work through them easier, and close them sooner.

  • The List all tabs button, which appears whenever the tabs bar overflows, opens a menu of all tabs in the current window. Keyboard shortcut: Ctrl+Shift+Tab.

    • Each menu item can be right-clicked to open its own tab context menu. This is as close as Firefox currently gets to native vertical tabs!

    • That said, Winger works well with vertical tabs, tab groups and other addons if you'd like the extra tab management help within individual windows.

  • Search tabs natively in the address bar by typing % Space followed by your search terms.

    • If you need more powerful ways to search and grab tabs, there are various addons to help (e.g. Select Tabs, a simple bulk tab selector I made).

  • Use an automatic tab discarding (a.k.a. unloading/suspending) addon to minimise system resouce usage on less-visited tabs and windows. Recommended for the freedom to spawn as many windows as you need for granular tab organisation.

  • If you are not ready to permanently close windows you hardly use, consider stashing them.
